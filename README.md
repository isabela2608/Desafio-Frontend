# Desafio Frontend

Projeto desenvolvido com React e Vite para consumir dados da Fake Store API e exibir produtos em uma interface web.

## Sprint 1 — Estrutura Base e Listagem

### Objetivo
Configuraração do projeto e listar os dados da API.

### Funcionalidades implementadas
- Projeto iniciado com Vite.
- Consumo da Fake Store API utilizando Fetch.
- Criação de um Header com logo e menu simples.
- Grid responsivo para exibição dos produtos.
- Componente de Card de Produto exibindo:
  - Imagem
  - Título
  - Preço em reais (R$)

### Tecnologias utilizadas
- React
- Vite
- JavaScript
- CSS Modules
- Fake Store API
